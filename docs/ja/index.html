SharePointの隠れたストレージ問題

Microsoft 365はデフォルトですべてのファイルのすべてのバージョンを保存します。時間の経過とともに、バージョン履歴がストレージクォータの30〜70%を気づかないうちに消費します。

原因不明のクォータ超過

管理センターはストレージが満杯と表示しますが、誰もその理由がわかりません。原因は、もう使われていないファイルの数千もの蓄積されたバージョンです。

毎月増加するコスト

追加ストレージは$0.20 USD/GB/月。不要なバージョン1TB = 年間$2,400 USD — 完全に回避可能です。

大規模対応のネイティブツールなし

Microsoftはテナントレベルでバージョンを一括クリーンアップする方法を提供していません。サイトごとの手動クリーンアップは大規模組織では現実的ではありません。
