What is this?

This page shows real-time aggregated statistics from organizations using SPO Version Management to optimize their SharePoint Online storage consumption.

All data is completely anonymous — no tenant names, site URLs, user identities, or document content is ever collected or transmitted.

Privacy Guarantees

  • Tenant identity is one-way hashed (SHA-256) — cannot be reversed
  • No site URLs, file names, or user data is transmitted
  • Only aggregate numbers: bytes freed, versions deleted, sites count
  • Telemetry is opt-in and can be disabled at any time
  • Source code is open — verify exactly what is sent

How to Participate

To contribute your anonymous savings data to the worldwide counter:

  • Enable telemetry in Settings (Dashboard or GUI App)
  • Set your telemetry endpoint in config\AppPaths.json
  • Run optimization sessions as normal — stats are sent automatically
